Perbedaan Block vs Inline

Block-level elements

<p> dan <div> akan mengisi lebar penuh dan mulai di baris baru.

Ini paragraf (p) sebagai block.

Ini <div> juga block.

Inline elements

Elemen inline tidak memulai baris baru dan lebarnya sesuai konten:

<span> <a> link <strong> <em>

<div> sebagai Container

Ini contoh <div> yang berfungsi sebagai container. Lebar dibatasi dan dipusatkan dengan margin: auto.

Class & ID

Artikel Mini: Perbedaan Class vs ID

Class dapat dipakai banyak elemen (contoh class city, main pada artikel ini), sedangkan ID harus unik dalam satu halaman (contoh #judulUnik di heading).

.city .main #judulUnik

Coba klik menuju bagian Iframe (contoh bookmark link ke id lain).

Menyematkan Halaman dengan <iframe>

Iframe di atas menggunakan attribute title, ukuran diatur lewat CSS, dan tanpa border.

Tips Belajar Cepat

  • Gunakan header, nav, section, article, aside, footer untuk struktur yang jelas.
  • Bedakan block vs inline saat menyusun layout.
  • Manfaatkan class untuk gaya berulang, id untuk identitas unik.
  • Sematkan materi referensi menggunakan iframe bila perlu.
